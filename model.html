Machine Learning Model

The goal of the model was to predict shelter animal adoptions. Although there were other outcomes listed in the data set, all other outcomes were changed to "not adopted" to simplify the model with consideration to what information would be most helpful for a shelter. The following variables were used in the creation of the model:

  1. Primary breed
  2. Sex
  3. Age
  4. Reason for intake
  5. Condition at intake
  6. Animal type
  7. Length of stay in shelter


The data was numerically encoded using the Pandas "get dummies" function. I chose to use a Random Forest for the model given the number of different variables included in the dataset. A grid search was then performed to configure the optimal parameters. The classifcation report created post-grid search is shown below:



Although the accuracy of the model is not extremely high, the model could still be useful for shelters and rescue operations trying to determine the outcomes of rescued animals.
